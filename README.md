# DiscordAccountExporter

A safe Discord tool to export and backup your account.

## Features
* Exports all private dms and group messages
* Exports user info

## Installation

```bash
npm install
node app.js
```

## Usage

1. Enter your Discord token
2. Wait for the app to finish exporting
